Symmetric Distribution

  • Symmetric Distribution:

    A symmetric distribution is a probability distribution where all values are evenly distributed around the mean. The most well-known symmetric distribution is the normal distribution, whose graph shows the well-known Bell Curve. In a symmetric distribution the mean, median and mode are all equal.

  • Skewness:

    Skewness is a way to describe the asymmetry of a distribution. This value can be negative, zero, or positive. In a symmetric distribution the skewness equals 0.

    • left skewed = negative skewness = mean < median
    • right skewed = positive skewness = median < mean

  • Bimodal:

    A bimodal distribution is a distribution that has two peaks. A bimodal distribution can be symmetric if the two peaks are of equal height. For a symmetric bimodal distributions the mean and the median are equal, while the mode is located in the two peaks.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.stats import gaussian_kde, skewnorm

def get_mode(arr):
    mode = pd.Series(np.round(arr, 1)).mode()
    if len(mode) == len(arr):
        return None
    return [i for i in mode]

def kde(data):
    x = np.linspace(min(data), max(data), 100)
    density = gaussian_kde(data)
    y = density(x)
    return (x, y)

normal = np.random.normal(size=1000)
right_skewed = np.random.exponential(size=1000)
left_skewed = skewnorm.rvs(a=-10, size=1000)
bimodal = np.concat([
    np.random.normal(loc=5, size=500),
    np.random.normal(loc=10, size=500)
])

def plot_dist(data):
    x, y = kde(data)
    plt.hist(data, bins=20, color="lightgray", ec="black", density=True, alpha=0.5)
    plt.plot(x, y)
    plt.axvline(np.mean(data), c="red", linestyle="dashed", label="mean")
    plt.axvline(np.median(data), c="orange", linestyle="dashed", label="median")
    if get_mode(data):
        for i in get_mode(data):
            plt.axvline(i, c="green", linestyle="dashed", label="mode")
    plt.legend()

dist_dict = {"normal": normal,
             "positively skewed": right_skewed,
             "negatively skewed": left_skewed,
             "bimodal": bimodal}

plt.figure(figsize=(8, 20))
idx = 1
for k, v in dist_dict.items():
    plt.subplot(4, 1, idx)
    plot_dist(v)
    plt.title(k)
    idx += 1
plt.show()
