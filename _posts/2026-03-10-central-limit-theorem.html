Central Limit Theorem¶

The Central Limit Theorem (CLT) states that a distribution of sample means approximates to normal as the sample size becomes large, no matter the population's original distribution. The sampling distribution's mean becomes equal to the population mean, and its variance equal to the population variance divided by the sample size.

$\bar{X}\sim\mathcal{N}(\mu,\frac{\sigma^2}{n})$

CLT is valid for the following conditions:

  • The drawing of a sample from the population should be random.
  • The drawing of a sample should be independent of each other.
  • Sample size shouldn't be more than 10% of the population when sampling is done without replacement.
  • Sample size should be sufficiently large (usually n > 30).
  • CLT only holds for a population with finite variance.

import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
from scipy.stats import gaussian_kde
from scipy.stats import skew

def sample(x, n=50):
    copy = x.copy()
    np.random.shuffle(copy)
    return copy[:n]

plt.figure(figsize=(8, 5))

means = np.array([np.mean(sample(a)) for _ in range(100)])

The standard deviation of the sampling distribution is also called the Standard Error.

Standard Error: $\sigma_{\bar{X}}=\frac{\sigma}{\sqrt{n}}$

def density_plot(data):
    graph = sns.kdeplot(data)
    plt.axvline(np.mean(data), c="black", linestyle="dashed", label=f"mean={np.mean(data):.2f}")
    kde = gaussian_kde(data)
    x = np.linspace(graph.get_xlim()[0], graph.get_xlim()[1], 100)
    y = kde(x)
    low, high = np.mean(data) - np.std(data), np.mean(data) + np.std(data)
    xy = [(x, y) for x, y in zip(x, y) if low <= x <= high]
    plt.fill_between([i[0] for i in xy], [i[1] for i in xy], color="lightblue")
    plt.errorbar(np.mean(data), 0.01, xerr=np.std(data), marker="s", capsize=5, label=f"std={np.std(data):.2f}")

Standardization:¶

Now that we have a normal distribution we can easily standardize it. The formula is: $Z=\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}$.

Standardization transforms a sample mean into a standard normal variable, facilitating statistical inference.

Note how the exponential distribution that we started with can be converted into a normal distribution, and from there into a standard normal distribution, just by drawing large enough samples.

Conclusion¶

The Central Limit Theorem is crucial in statistics, as it allows the use of statistical methods that rely on a normal distribution, even when the data is not normally distributed. As long as the sample size is large enough, we can always assume a normal distribution of sample means.
